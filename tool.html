評価ツールの説明

ここではアクセシビリティおよびバリデーションに利用したツールについての説明を行います。


The W3C Markup Validation Service

The W3C Markup Validation Serviceは、W3Cが提供しているHTMLの検証ツールです。URLを入力し、Checkをクリックすると、errorやwarningの数、それらが検出された行などが表示されます。

W3C CSS 検証サービス

W3C CSS 検証サービスは、W3Cが提供しているCSSの検証ツールです。URLを入力し、「検証する」をクリックすると、エラーや警告の数、それらが検出された行などが表示されます。また、このツールは日本語に対応しています。
